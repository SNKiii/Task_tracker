Базовые команды для работы с Task tracker:
-- /add - добавление задачи. Требования:
    1) Обязательные поля - name, body, importanceLevel, dueDate.
        - name - строка
        - body - цели и суть задачи
        - importanceLevel - степень важности (от 1 до 10). Принимается только цифры (0-9)
        - dueDate - строгая дата для указания срока выполнения задачи
            (При неопределённости в дате
            или поле останется пустым,
            будет указан как год для выполнения)

-- /show - просмотр задач\задачи из уже добавленных
    =================ПРЕФИКСЫ======================
    1) (-a) - префикс для просмотра всех задач
        Пример: /show -a

    2) (-i) - префикс для просмотра определенной задачи
        1) Обязательные поля - id, используя цифры (0-9)

    3) (-in) - префикс для просмотра определенной задачи по названию
         1) Обязательные поля - name

    4) (-l) - префикс для просмотра важных задач (уровень от 6 и выше)

    5) (-li) - для просмотра задач с определенным уровнем важности. Требования:
        1) Обязательные поля - importanceLevel (Выводит задачи с указанным уровнем)

    6) (-si) - просмотр стадии определенной задачи по id

-- /update - изменение полей задачи
    =================ПРЕФИКСЫ======================
    1) (-s) - префикс для изменения стадии задачи. Требования:
        1) Обязательные поля - stage( число от 1 до 5), id

    2) (-dt) - префикс для изменения ограничений по дате. Требования:
        1) Обязательные поля - dueDate (Формат: YYYY-MM-DD), id или name

-- /delete - удаление задач
    =================ПРЕФИКСЫ======================
    1) (-a) - удаление всех задач. (Используйте с осторожностью,
              так как незаписанные задачи не сохранятся при перезапуске)

    2) (-aR) - полное удаление задач. (Опасная команда, которая удалит все задачи,
               которые когда-либо были записаны)

    3) (-i) - удаление задачи по id в текущей памяти приложения.

    4) (-iR) - удаление задачи из внешней памяти, без возможности восстановления.
